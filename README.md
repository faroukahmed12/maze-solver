  Maze Solver using AI Algorithms

 Project Overview

This project implements an intelligent agent that solves a maze using Artificial Intelligence search algorithms. The system finds a path from a start point to a goal point while avoiding obstacles.

The project demonstrates core AI concepts such as problem-solving, search strategies, and rational decision-making.

---

 Objectives

* Apply AI search algorithms to solve real-world problems
* Compare different search strategies
* Visualize the solution using a graphical interface (GUI)
* Understand how an intelligent agent operates in an environment

---

 Algorithms Implemented

 Breadth-First Search (BFS)

* Explores nodes level by level
* Guarantees the shortest path
* Slower in large spaces

 Depth-First Search (DFS)

* Explores deeply into the maze
* Does not guarantee optimal solution

 A* Search Algorithm

* Uses heuristic (Manhattan Distance)
* Combines cost + estimated distance
* Fast and optimal
